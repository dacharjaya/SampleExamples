Input text to test Palindrome.

Palindrome: true

Steps to do

  1. lowerCase the input and replace regular expression with /[^A-Za-z0-9]/g or /[W]/g
  2. reverse the input split('').reverse('').join('')
  3. match the input and reversed input

Actual Implementation

$("#input").keyup(function() {
  var re = /[^A-Za-z0-9]/g;
  var reInput = $("#input").val().toLowerCase().replace(re,'');
  var reverseReInput = reInput.split('').reverse().join('');

  $('#status').text(reInput === reverseReInput);
});
